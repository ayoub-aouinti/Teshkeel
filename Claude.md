## Règles absolues
- Ne jamais éditer les fichiers .env
- Toujours écrire les tests avant de dire "done"
- Fichiers < 150 lignes, découper si nécessaire

## Style de code
- Nommage en camelCase pour les variables
- PascalCase pour les composants React
